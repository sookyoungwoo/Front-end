선택자는 웹문서의 어느 부분에 스타일을 적용할지 알려주는 것
* : 전체선택자
1. HTML 요소 선택자   
-CSS를 적용할 대상으로 HTML 요소의 이름을 직접 선택  
2. 아이디(id) 선택자
-CSS를 적용할 대상으로 특정 요소를 선택할 때 사용
3. 클래스(class) 선택자  
-특정 집단을 클래스라고 하며, 클래스 선택자는 같은 클래스의
여러 요소를 한 번에 선택할 떄 사용 
4. 그룹(group) 선택자
-위의 선택자를 같이 사용하고자 할 때 사용. 그룹 선택자는
쉽표로 구분하여 연결

스타일 상속
: 태그의 포함 관계에 따라 부모요소의 스타일을 자식요소로 전달
캐스케이딩 Style:우선순위가 위에서 아래 계단식으로 적용됨

1)스타일 우선순위
사용자 스타일 -> 제작자 스타일 -> 브라우저 기본 스타일

2) 소스 작성 순서에 따른 순위
: 나중에 작성한 스타일이 이전 스타일을 덮어씀
