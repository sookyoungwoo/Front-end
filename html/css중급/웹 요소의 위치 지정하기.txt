1. position 속성 : 웹 요소의 배치 방법 지정
1)static : 문서의 흐름에 맞춰 배치(기본값)
2)relative : 위치값을 지정하며 흐름에 맞춰 배치
3)absolute : relative 값을 사용한 상위 요소를 기준으로 위치 지정
**position : absolute 사용시 부모 요소에 position:relative를 꼭 적용
4)fixed : 브라우저 창을 기준으로 위치를 지정

2. left, right, top, bottom : 웹 요소의 위치 지정
1)left : 기준 위치와 요소 사이에 왼쪽으로 얼마나 떨어져 있는지 지정
2)right : 기준 위치와 요소 사이에 오른쪽으로 얼마나 떨어져 있는지 지정
2)top : 기준 위치와 요소 사이에 위쪽으로 얼마나 떨어져 있는지 지정
2)bottom : 기준 위치와 요소 사이에 아래쪽으로 얼마나 떨어져 있는지 지정
